cDock

Dock customization for OS X 10.9 and 10.10

Download   GitHub      Issues   

Basic Information:

  • cDock installs part of EasySIMBL if it is not already installed.
  • cDock works for both the Bottom Dock and Side dock.
  • The install/uninstall is for the current user only.
  • Dock theming requires OSX 10.9 and above

System Information:

  • Working on:
    • 10.9 to 10.9.5 (13F34)
    • 10.10 to 10.10.3 (14D131)

  • Not working:
    • 10.10.4 (14E36b)
    • 10.11 (15A204h)
