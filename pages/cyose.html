Classic Folders

Replace newer OSX folder icons with Mavericks folder icons

Download   GitHub      Issues   

Information:

  • Working on:
    • 10.10 to 10.10.4 (14E36b)
    • 10.11 to 10.11 (15A204h)

  • Not working:
    • None

How to use:

  1. Download the latest release
  2. Unzip the download
  3. Open classic_folders.command
  4. Follow instrutions printed to terminal window
