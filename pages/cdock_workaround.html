cDock 10.10.4 Workaround

How to:

  1. Download SIMBL 0.9.9
  2. Unzip and Install SIMBL
  3. Copy cDock (and/or ColorfulSidebar) bundles to /Library/Application Support/SIMBL/Plugins/
    4. # cDock bundle
$ cp -r /Applications/cDock.app/Contents/Resources/bundles/cDock.bundle /Library/"Application Support"/SIMBL/Plugins/cDock.bundle

# ColorfulSidebar bundle
$ cp -r /Applications/cDock.app/Contents/Resources/bundles/ColorfulSidebar.bundle /Library/"Application Support"/SIMBL/Plugins/ColorfulSidebar.bundle
  4. Open Script Editor
  5. Paste in this
    6. try
	tell application "Dock"
		inject SIMBL into Snow Leopard
	end tell
end try
try
	tell application "Finder"
		inject SIMBL into Snow Leopard
	end tell
end try
  6. Run script
  7. Save script
  8. Add saved script to your login items
