Dark Boot

Give your mac the new Dark Boot Screen on OS X 10.10 and 10.11

Download GitHub Issues

Information:

  • Working on 10.10 to 10.11
  • This may break in the future
  • This should work on all Macs. Submit a report here if it does not

How to use:

  1. Download the latest release
  2. Unzip the download
  3. Open the Dark Boot app
  4. Select the color you would like your boot screen to be and press OK
  5. Enter your admin password and press OK
  6. Reboot twice for changes to take effect

Warning:

This app modifies the systems boot.efi which has the potential to prevent the system from booting.

If this happens you can boot into recovery and entering the following command in terminal:

cd /System/Library/CoreServices
mv ./boot.efi ./boot.efi.bak
mv ./boot_stock.efi ./boot.efi
chmod 644 ./boot.efi
chown root:wheel ./boot.efi
chflags uchg ./boot.efi
reboot
