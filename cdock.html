cDock

Dock customization for OS X 10.9 and above

  • Theme your dock from a selection of designs.
  • Hide annoying dock background.
  • Give dock additional functionality.
  • Adjust hidden dock settings.

Information:

  • cDock installs SIMBL if it is not already installed.
  • cDock works for all orientations of the Dock.
  • Themes are applied for the current user only.

El Capitan Compatibility:

  • Colored Finder sidebar is no longer supported.
  • Builds from version 9.0 of cDock are compatible with El Capitan.
  • With the introduction of the new System Integrity Protection feature in OS X, additional setup steps are required to allow Dock theming.

    Full instructions are available at Setting up System Item Control in OS X El Capitan following these steps will allow cDock to theme the Dock, and still have System Integrity Protection Enabled.
