cDock

Dock customization for OS X 10.9, 10.10, and 10.11

Download  GitHub   Issues 

Previous Next

Basic Information:

  • cDock may break in the future, and I'm not obligated to fix it, be careful when updating.
  • cDock requires that System Integrity Protection be disabled for 1st install on OS X 10.11.
  • cDock installs SIMBL if it is not already installed.
  • cDock works for both the Bottom Dock and Side dock.
  • The install/uninstall is for the current user only.
  • Dock theming requires OSX 10.9 and above

cDock is maintained by w0lfschild.
